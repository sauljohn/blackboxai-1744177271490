# Lamps & Lighting Replica

## Project Overview
Lamps & Lighting Replica is an interactive web application that showcases a variety of lamps and lighting products. Designed using modern web development practices, it features a responsive layout, appealing design, and dynamic content generation. This project aims to provide users an easy way to browse and learn about different lighting options available, enhancing the overall shopping experience.

## Installation
To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd lamps-lighting-replica
   ```

2. **Open the project in your web browser:**
   This project does not have any server-side components, so you can simply open the `index.html` file in your preferred web browser.

## Usage
Once you open the `index.html` file in a web browser, you will see a beautifully designed page with a header, product grid, and footer. The products are dynamically rendered using JavaScript, allowing immediate feedback for interactions such as adding products to the cart.

## Features
- **Responsive Design:** The layout adjusts to different screen sizes to ensure a great user experience on mobile and desktop devices.
- **Dynamic Product Rendering:** Products are defined in a JavaScript array and displayed using a render function, ensuring easy updates and modifications.
- **Intuitive Navigation:** Users can navigate seamlessly through links for home, lamps, lighting, and contact information.
- **Customer Interaction:** Features icons for search, user account, and shopping cart in the header.

## Dependencies
This project utilizes the following libraries:
- **Tailwind CSS:** For styling and responsive design, loaded via CDN.
- **Font Awesome:** For icons, loaded via CDN.

There are no additional dependencies listed in a `package.json` file as this project primarily consists of static HTML, CSS, and JavaScript files.

## Project Structure
The project has a simple structure consisting of the following files:

```
lamps-lighting-replica/
├── index.html
```

- **`index.html`**: The main HTML file that contains the entire structure and layout of the web page, including header, product grid, and footer, as well as the embedded JavaScript for rendering the products dynamically.
